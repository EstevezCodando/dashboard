Bem-vindo à API BackEnd do SAP para Dashboard!

Use os endpoints abaixo para acessar os dados necessários.

Lista de Endpoints

  • /api/lotes

    Retorna todos os lotes disponíveis no banco de dados, incluindo seus IDs e nomes.

  • /api/subfases

    Retorna todas as subfases disponíveis no banco de dados, incluindo seus IDs e nomes.

  • /api/materialized_views

    Lista todas as materialized views disponíveis no schema acompanhamento.

  • /api/lotes_subfases

    Retorna os lotes e as subfases associadas, com nomes descritivos e as views correspondentes.

  • /api/view/{view_name}

    Retorna os dados de uma materialized view específica. Substitua {view_name} pelo nome da view desejada.

  • /api/dados_views

    Retorna os dados de todas as materialized views no schema acompanhamento.
